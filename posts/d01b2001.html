十、Spring 整合 MyBatis

  • 原始dao开发【2种】
  • mapper代理开发【2种】

1、整合思路

  • 需要spring通过单例方式管理SqlSessionFactory。
  • spring和mybatis整合生成代理对象，使用SqlSessionFactory创建SqlSession。（spring和mybatis整合自动完成）
  • 持久层的mapper都需要由spring进行管理。

  • 新建 StudentMapperImpl.java 继承于 SqlSessionDaoSupport

    • 注：必须要把SqlSessionDaoSupport中的sqlSessionFactory属性赋值
    @Repository("studentMapper2")
public class StudentMapperImpl2 extends SqlSessionDaoSupport implements StudentMapper {

	@Autowired
	public void setSqlSessionFactory(SqlSessionFactory sqlSessionFactory) {
		super.setSqlSessionFactory(sqlSessionFactory);
	}
	@Override
	public Student findById(int id) {
		SqlSession session = this.getSqlSession(); 
		Student stu = session.selectOne("net.wanho.mapper.StudentMapper.findById",id);
		return stu;
	}

}
