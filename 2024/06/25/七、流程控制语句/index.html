七、流程控制语句

Python

七、流程控制语句

1、选择结构

单分支选择结构

if 语句

if 语句是用来进行判断的，其使用格式如下：

if 要判断的条件:
    条件成立时要做的事情

注意：代码的缩进为一个 tab 键，或者 4 个空格

双分支选择结构

if-else 语句

if 和 else 语句以及各自的缩进部分共同是一个 完整的代码块

使用格式
if 条件:
  满足条件时的操作
else:
  不满足条件时的操作

weather = '晴天'
if weather == '下雨':
    print('出门要带伞') # 缩进
    print(111)
else:
    print('戴个帽子')

多分支选择结构

if-elif-else 语句

使用 if 可以 判断条件；使用 else 可以处理 条件不成立 的情况。

但是，如果希望 再增加一些条件，条件不同，需要执行的代码也不同 时，就可以使用 elif。

使用格式
if xxx1:
  事情1
elif xxx2:
  事情2
elif xxx3:
  事情3
else:
  不满足条件时的操作

# bmi计算 bmi =w/(h*h)
w = float(input('请输入你的体重，单位kg：'))
h = float(input('请输入你的身高，单位米：'))
bmi = w / (h * h)
if bmi < 18.5:
    print('多吃一点才健康')
elif bmi < 23.9:
    print('你的体型非常的标准')
else:
    print('适当的可以多运动一下')

嵌套选择

在开发中，使用 if 进行条件判断，如果希望 在条件成立的执行语句中 再 增加条件判断，就可以使用 if 的嵌套

if 的嵌套 的应用场景就是：在之前条件满足的前提下，再增加额外的判断

if 的嵌套 的语法格式，除了缩进之外 和之前的没有区别

if 嵌套

格式
if 条件1:
    满足条件1 做的事情1
    满足条件1 做的事情2
    if 条件2:
        满足条件2 做的事情1
        满足条件2 做的事情2
  • 外层的 if 判断，也可以是 if-else
  • 内层的 if 判断，也可以是 if-else
  • 根据实际开发的情况，进行选择

扩展

区间比较

在Python中允许类似于 5 < a < 10 的写法，这种写法在很多编程语言里都是不允许的，需要使用逻辑与运算符进行连接。

a = 10
1 < a < 20  # True
11 < a < 20  # False

三元运算符

格式

  • 变量 = 值1 if 条件 else 值2

  • 当条件成立时把 值1 赋值给变量，条件不成立时把 值2 赋值给变量，其实就是对 if…else 语句的一种简写。

Case
a = 10
b = 30
if a > b:
    c = a
else:
    c = b
print('a和b两个数的较大值是%d'%c)

以上的 if...else 代码可以简写成以下格式:

a = 10
b = 30
c = a if a > b else b
print('a和b两个数的较大值是%d' % c)

自动类型转换

if 语句需要一个判断条件，这个判断条件的结果需要一个布尔值。如果此时输入的判断条件不是一个布尔值，在代码执行的过程中，会将这个值自动转换成为一个布尔值。

if 1 < 2:
    print('1小于2')

if "hello":
    print("这段代码是否会输出呢？")

提示：在Python中，转换成为布尔值时，只有 0,"",'',None,(),{},[] 会被转换成为False,其他都会被转换成为True

2、循环语句

循环的作用就是让 指定的代码 重复的执行。

while 循环

概念

while 循环最常用的应用场景就是 让执行的代码按照指定的次数 重复 执行。

while 判断条件:
    代码块

while & if 语句区别

我们发现，while 循环的语法结构和 if 语句的结构很相像，那么这两个之间有什么区别呢？

  • if 语句只会执行一次判断条件，条件成立以后，只会执行一次代码块；
  • while 语句，会先执行一次判断条件，如果条件成立，会执行一次代码块，代码块执行完成以后，会再次判断条件语句，如果还成立，将会再执行代码语句… 直到最后条件不成立。

输出十遍 hello world

i = 1
while i <= 10:
    print('hello world')
    i += 1  # 一定要记得让 i += 1, 否则会造成死循环

死循环

概念：条件始终为真的循环称为死循环
在循环语句中，表达式永远为真，致循环持续执行，程序无法终止

while True:
    print("放我出去")

除了 while 循环以外，for 可以完成循环的功能。

for 循环

在 Python 中 for 循环可以遍历任何序列的项目，如一个列表或者一个字符串等。

格式

for 临时变量 in 列表或者字符串等可迭代对象:
    循环满足条件时执行的代码
  • 注意: in 的后面必须要是一个可迭代对象!!!
  • 目前接触的可迭代对象: 字符串、列表、字典、元组、集合、range
# 遍历字符串
for s in "hello":
    print(s)
    
"""
h
e
l
l
o
"""

# 打印数字
for i in range(5):
    print(i)

"""
0
1
2
3
4
"""

range() 函数

range()函数可以用来创建一个数字序列，常与for循环结合使用来重复执行代码块指定的次数。

range() 可以传递三个参数，分别表示 起始、结束和步长。

  • range(101)：可以用来产生0到100范围的整数，需要注意的是取不到101。
  • range(1, 101)：可以用来产生1到100范围的整数，相当于前面是闭区间后面是开区间。
  • range(1, 101, 2)：可以用来产生1到100的奇数，其中2是步长，即每次数值递增的值。
  • range(100, 0, -2)：可以用来产生100到1的偶数，其中-2是步长，即每次数字递减的值。

break 和 continue

break 和 continue 都用在循环语句里，用来控制循环。

break 的使用

作用：退出循环
注意：只能跳出距离最近的for或者while循环

i = 0
while i < 10:
    i += 1
    if i == 7:
        break
    print(i)
    
"""
1
2
3
4
5
6
"""

在循环过程中，如果 某一个条件满足后，不 再希望 循环继续执行，可以使用 break 退出循环。

continue 的使用

作用：跳过本次循环后面的剩余语句，然后继续下一次循环
注意：只能跳过距离最近的for或者while循环

i = 1
while i <= 10:
    if i == 7:
        continue
    print(i)
    i += 1
    
"""
1
2
3
4
5
6
8
9
10
"""

与else结合

没有通过 break 退出循环，循环结束后，会执行的代码

n = 5
for i in range(2,n):
    if n%i==0:
        print(n,'不是质数')
        break
else:
    print(n,'是质数')

注意点

  1. 在 Python 中，break 和 continue 只能用于循环语句中。
  2. break 和 continue 在嵌套循环中使用时，只对最内层循环有效。

pass

作用：当语句要求不希望任何命令或代码来执行时使用。

说明：pass 语句表示一个空操作，在执行时没有任何的响应，pass 的位置最终应该有代码来执行，只不过暂时写不出来。

可以使用在流程控制和循环语句中。

if 1:
    pass

嵌套循环

类似 if 的嵌套，while 嵌套就是：while 里面还有 while

while 嵌套的格式

while 条件1:
      外循环代码块
      while 条件2:
            内循环代码块

内循环一旦开始，则要执行到条件2不满足，才会继续执行外循环的代码。

i = 0  # 内外循环的控制变量不能一样
while i < 5:
    j = 0  # 内循环的控制变量必须要在外循环里初始化
    while j < 5:
        print('j= %d' % j)
        j += 1
    print('i= %d' % i)
    i += 1

else 语法

在 Python 中，循环语句还可以和 else 语句联合使用。

while…else…

while 判断条件:
  条件成立时，循环体代码
else:
  条件不成立时，执行的代码

for…else…

for x in 序列：
   循环体代码 
else:
   循环完所有序列，执行的代码

else 作用

i = 0
while i < 5:
    print("循环条件成立时执行的代码")
    i += 1
else:
    print('循环条件不成立时执行的代码')
    
"""
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件不成立时执行的代码
"""

从上述结构中，我们可以看出，在非死循环中，正常情况下 else 里的语句都是会被执行的。那这个else语句到底有什么作用呢？

一般情况下，循环语句和 else 共同出现时，在循环语句里都会配合 break 语句来使用。查看下面的代码:

i = 0
while i < 10:
    if i == 5:
        break
    print('循环条件成立时执行的代码')
    i += 1
else:
    print('循环条件不成立时执行的代码')
    
"""
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
循环条件成立时执行的代码
"""

执行后发现 else 没有被执行，只要循环没有顺利完成，中间被 break 中断了则 else 里面的内容就不会执行。
